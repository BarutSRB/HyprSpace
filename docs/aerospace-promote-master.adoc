:manpurpose: Promote the focused window to be the master window

// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace promote-master [-h|--help] [--window-id <window-id>]

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Swaps the focused window with the current master window (the first window in the master layout).
This command only works when the parent container is using the `master` layout.

If the focused window is already the master window, the command will fail with an error message.

`aerospace promote-master`

cmd-shift-p = 'promote-master'
